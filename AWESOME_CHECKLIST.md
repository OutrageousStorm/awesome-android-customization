# Awesome Android Customization — Tracker

This file tracks the most impactful tools and ROMs on this awesome list.

## Top Tools by Impact

### Tier 1 (Most-Used)
- **LineageOS** — Most popular custom ROM, massive community
- **Magisk** — De-facto root standard, systemless approach
- **TWRP** — Custom recovery gold standard
- **Frida** — Dynamic instrumentation, security research
- **NetGuard** — No-root firewall, per-app network control

### Tier 2 (Specialized)
- **GrapheneOS** — Hardest target for privacy
- **KernelSU** — Kernel-level root alternative to Magisk
- **OrangeFox Recovery** — Modern TWRP alternative
- **PCAPdroid** — Network traffic capture, no root
- **App Manager** — Full app control, no root needed

### Tier 3 (Niche/Emerging)
- **CalyxOS** — MicroG-based privacy ROM
- **Shizuku** — ADB-level access without USB
- **Obtainium** — Direct GitHub app installer
- **F-Droid** — FOSS app store ecosystem
