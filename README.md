# Awesome Android Customization ✨

A curated list of the best Android customization tools, ROMs, mods, and tweaks.

## 📱 Custom ROMs

### Pixel-Focused
- **[GrapheneOS](https://grapheneos.org)** — Maximum privacy, Pixel-only. Hardware-backed security.
- **[CalyxOS](https://calyxos.org)** — Privacy + usability, Pixel + some others. Built-in microG.
- **[DivestOS](https://divestos.org)** — 200+ devices. Optional microG. Very lightweight.

### Universal AOSP
- **[LineageOS](https://lineageos.org)** — Most devices, clean AOSP, great community.
- **[crDroid](https://crdroid.net)** — Heavy customization, snappy performance.
- **[Evolution X](https://evolution-x.org)** — Pixel-inspired UI, feature-rich.
- **[Pixel Experience](https://pixelexperience.org)** — Pure Pixel ROM for any device.

## 🛠️ Modding Tools

### Root & Privilege Escalation
- **[Magisk](https://magisk.me)** — Systemless root with Zygisk framework and extensive module ecosystem.
- **[KernelSU](https://kernelsu.org)** — Kernel-level root with better evasion and GKI support.
- **[Shizuku](https://shizuku.rikka.app)** — ADB-level access without root — extremely useful for privacy/tweaks.

### Custom Recovery
- **[TWRP](https://twrp.me)** — Most compatible, best documentation. Huge device support.
- **[OrangeFox](https://orangefox.download)** — Modern UI, active development.

### APK Tools
- **[jadx](https://github.com/skylot/jadx)** — Decompile APKs to readable Java.
- **[apktool](https://apktool.org)** — Disassemble and repackage APKs in Smali.
- **[Frida](https://frida.re)** — Dynamic instrumentation for runtime hooking and analysis.

## 🎨 Customization

### Launchers
- **[Lawnchair](https://lawnchair.app)** — Pixel launcher fork with heavy customization.
- **[Niagara](https://niagara.app)** — Radical UI redesign for speed.
- **[Evie](https://play.google.com/store/apps/details?id=is.shortcut)** — Minimal, fast, gesture-driven.
- **[Kvaesitso](https://kvaesitso.mm20.de)** — FOSS, highly customizable.

### Icon Packs
- **[Arcticons](https://play.google.com/store/apps/details?id=com.donnnno.arcticons)** — Gorgeous minimalist icons.
- **[Niksya](https://www.niksyaicons.com)** — Modern, colorful, well-designed.

### Theming (Xposed/LSPosed)
- **[Iconify](https://github.com/Mahmud0808/Iconify)** — Full system UI theming via LSPosed.
- **[GravityBox](https://github.com/C3C0/GravityBox)** — Swiss army knife of tweaks.

## 🔒 Privacy & Security

### VPN & Network
- **[Mullvad VPN](https://mullvad.net)** — Anonymous, accepts cash/crypto. No account needed.
- **[Orbot](https://orbot.app)** — Tor proxy for Android — use via any app with VPN.

### Keyboards & Input
- **[FUTO Keyboard](https://keyboard.futo.org)** — No network access, works fully offline.
- **[OpenBoard](https://github.com/openboard-team/openboard)** — FOSS, privacy-respecting.

### Password Managers
- **[Bitwarden](https://bitwarden.com)** — Open source, sync, browser integration.

### Authenticators
- **[Aegis](https://getaegis.app)** — FOSS TOTP with encrypted backup.
