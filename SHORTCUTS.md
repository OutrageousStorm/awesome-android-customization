# 🎯 Android Shortcuts & Automation

Quick-access guide to the most useful customization tricks on this list.

## By use case

### "I want to customize my home screen"
→ **Lawnchair** (launcher) + **Lawnicons** (icons) + **Kwgt** (widgets)

### "I want to debloat / remove system apps"
→ **ADB** (see android-adb-cheatsheet) or **System App Remover** (Play Store)

### "I want to use Magisk modules"
→ Install **Magisk** → Open Manager → Modules → Search repo

### "I want custom ROMs"
→ **LineageOS** or **GrapheneOS** (see android-rom-guide)

### "I want Xposed hooks without root"
→ **LSPosed** (requires Magisk + Zygisk) or **Shizuku** (no root)
