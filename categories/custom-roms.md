## What to install first after ROM

1. **Magisk** — root (if needed)
2. **Play Integrity Fix** — Google app compatibility
3. **NetGuard** — per-app firewall
4. **Shizuku** — elevated ADB without root
5. **Shelter** — app cloning + work profile isolation
