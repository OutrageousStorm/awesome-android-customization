# Advanced Launchers

Launchers for power users with deep customization.

| Launcher | Customization | Theme Support | Required |
|----------|---------------|---------------|----------|
| **Lawnchair** | Extreme — icon packs, icon shape, fonts, grid | Full | Just icons |
| **KISS Launcher** | Minimal, search-focused — faster than stock | Theme colors | Search-based workflow |
| **Niagara Launcher** | Modern vertical layout, gesture-heavy | Limited (by design) | Gesture preference |
| **Action Launcher** | Quickdrawers, cling actions, customizable grid | Full + icon packs | Advanced customization |
| **Evie** | Search-first with AI suggestions | Minimal | Speed over looks |
| **Hyperion Launcher** | Pixel-like with unlimited customization | Full material theming | Pixel feel + customization |
| **Apex Launcher** | Heavy customization, gesture support | Full + icon packs | Many options |
| **Nova Launcher** | Balanced — powerful but usable | Full theming | Best balance |

## Power user workflows

- **Minimize home screen**: KISS Launcher (search everything)
- **Maximum customization**: Lawnchair (icon shape, grid, fonts, gestures)
- **Pixel experience on non-Pixel**: Hyperion Launcher
- **Gesture-driven**: Niagara (swipe up/down for apps)
- **Legacy devices**: Apex or Nova (lightweight)

## Icon Pack Compatibility

All support: [Linebit](https://play.google.com/store/apps/details?id=com.edzondavidson.linebit), [Breezy Icons](https://play.google.com/store/apps/details?id=com.luckyzyx.breezyicons), [Utopiox](https://play.google.com/store/apps/details?id=com.edzondavidson.linebit)
