# File Managers for Android

The best alternatives to the stock file manager.

| Manager | Source | Features |
|---------|--------|----------|
| **Material Files** | F-Droid | FOSS, clean UI, root support, archive handling |
| **MiXplorer** | XDA Forum | Most feature-rich — SMB, SFTP, FTP, cloud, dual pane |
| **Ghost Commander** | F-Droid / Play | Dual pane classic, lightweight, highly customizable |
| **Solid Explorer** | Play Store | Material Design, cloud integration, archive support |
| **Amaze** | F-Droid | FOSS, modern material design, fast |
| **Total Commander** | Play Store | Legacy but powerful, plugins for everything |

## Best for different needs

- **Most features**: MiXplorer (free, no bloat)
- **FOSS purist**: Material Files or Amaze
- **Cloud storage**: Solid Explorer
- **Network (SMB/SFTP)**: MiXplorer or Ghost Commander
