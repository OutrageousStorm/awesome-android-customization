# Android Keyboards

Custom keyboard apps and input methods.

## FOSS Keyboards

| Keyboard | Features | Link |
|----------|----------|------|
| **OpenBoard** | Open source, minimal, fast | [GitHub](https://github.com/openboard-team/openboard) |
| **FUTO Keyboard** | Privacy-first, no network access | [FUTO](https://keyboard.futo.org) |
| **HeliBoard** | Fork of OpenBoard, modern UI | [GitHub](https://github.com/Helium314/HeliBoard) |
| **FlorisBoard** | Material You, modern, extensible | [GitHub](https://github.com/florisboard/florisboard) |

## Features Comparison

| Feature | Gboard | OpenBoard | FUTO | HeliBoard |
|---------|--------|-----------|------|-----------|
| Material You | ✓ | Limited | ✓ | ✓ |
| No tracking | ✗ | ✓ | ✓ | ✓ |
| Themes | ✓ | ✓ | ✓ | ✓ |
| Clipboard access | ✓ | ✗ | ✗ | ✗ |
| Gesture typing | ✓ | ✓ | ✓ | ✓ |
| Offline | ✗ | ✓ | ✓ | ✓ |

## Tips

1. **Privacy**: Use FUTO, OpenBoard, or HeliBoard — no cloud, no tracking
2. **Customization**: Gboard for Google integration, FlorisBoard for Material You
3. **Minimal**: FUTO Keyboard has the smallest footprint
4. **One-handed**: Thumb-Key designed for thumbs
