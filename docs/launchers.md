# Android Launchers

Best launcher apps for Android customization.

## High-End Customization

| Launcher | Features | Price |
|----------|----------|-------|
| **Niagara** | Gesture-based, minimal UI, widget drawer | Free / Pro |
| **Nova** | Heavy customization, icon packs, grids | Free / Pro |
| **Pixel Launcher** | Stock Android, clean, integrates Google | Free |
| **Lawnchair** | AOSP-based, highly customizable | Free |
| **Kvaesitso** | Keyboard-first search launcher | Free |
