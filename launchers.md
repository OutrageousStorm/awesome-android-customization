# Best Android Launchers

| Launcher | Type | Stars | Notes |
|----------|------|-------|-------|
| **Lawnchair** | FOSS | ⭐⭐⭐⭐ | Material Design, Pixel Launcher base |
| **Nova Launcher** | Freemium | ⭐⭐⭐⭐ | Most customizable, gestures |
| **Microsoft Launcher** | Free | ⭐⭐⭐ | Tight Windows integration |
| **Niagara** | Premium | ⭐⭐⭐⭐ | Radical redesign, unique |
| **CPL Launcher** | FOSS | ⭐⭐⭐ | Simple, lightweight |
| **Olauncher** | FOSS | ⭐⭐⭐ | Minimalist, grid-based |
| **Hyperion Launcher** | Premium | ⭐⭐⭐ | Feature-rich |
| **Action Launcher** | Free | ⭐⭐⭐ | Quickdrawer, gestures |

## Recommendations by use case

**Best overall:** Lawnchair (FOSS) or Nova (feature-rich)  
**Minimalist:** Olauncher or Niagara  
**Power user:** Nova or Hyperion  
**Privacy-first:** Lawnchair or CPL Launcher  

## Tips
- Use Nova + Tasker for full automation
- Lawnchair + Icon packs for theming
- Use QuickSwitch (Magisk module) to swap Pixel Launcher with your choice
