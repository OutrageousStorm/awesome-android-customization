# Custom Launchers & Wallpaper Engines

## Popular launchers (rooted & non-rooted)

| Launcher | Rooting required | Key features | Link |
|----------|-----------------|--------------|------|
| **Nova Launcher** | No | Gesture control, icon packs | Play Store |
| **Lawnchair** | No | Google Launcher-based | [GitHub](https://github.com/LawnchairLauncher/lawnchair) |
| **KISS Launcher** | No | Minimal, fast | F-Droid |
| **Niagara Launcher** | No | Vertical layout, pocketable | Play Store |
| **Poco Launcher** | No | MIUI-style | [APK](https://www.apkmirror.com) |
| **Customized Pixel Launcher** | Yes (Magisk module) | Android 12+ Pixel UI | [XDA](https://forum.xda-developers.com) |

## Wallpaper engines

- **Muzei** (F-Droid) — rotating art wallpapers
- **Dweeb** (F-Droid) — live pixel art
- **Samsung Always On Display** (Magisk port)
- **Taiko no Tatsujin** live wallpaper (rhythm-based)

## Customization with root

With a Magisk module you can:
- Replace default Pixel Launcher with custom Lawnchair
- Inject custom icon packs system-wide
- Replace system fonts
- Custom status bar icons (via Iconify Xposed module)
