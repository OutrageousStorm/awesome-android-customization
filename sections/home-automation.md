# Home Automation & Android

Connecting Android devices to home automation and IoT.

## Home automation platforms

- **[Home Assistant](https://www.home-assistant.io/)** — open source, self-hosted, Lovelace UI
- **[OpenHAB](https://www.openhab.org/)** — vendor-agnostic, runs on Raspberry Pi
- **[Domoticz](https://domoticz.com/)** — lightweight, single-file install

## Android clients

| App | Purpose | Link |
|-----|---------|------|
| Home Assistant | Official HA app | F-Droid |
| OpenHAB | Official OpenHAB app | Play Store |
| Tasker + Join | Home automation scripting | [Tasker](https://tasker.joaoapps.com) |
| Macrodroid | Automation with triggers/actions | Play Store |
| AutoNotification | Notification-based automation | XDA |

## Android as a hub

Using an old Android tablet as a local hub (runs 24/7, LAN access):
1. Install Home Assistant Companion app
2. Enable device tracking + sensors
3. Use tablet's WiFi/Bluetooth for presence detection
4. Reduce brightness + auto-lock for battery preservation
