# CLI & Command-Line Tools

Terminal-based Android utilities for power users.

## ADB Automation
- [android-adb-cheatsheet](https://github.com/OutrageousStorm/android-adb-cheatsheet) — 150+ ADB commands reference
- [android-toolkit-scripts](https://github.com/OutrageousStorm/android-toolkit-scripts) — Python/Bash device automation
- [apk-patcher](https://github.com/OutrageousStorm/apk-patcher) — APK SSL pinning + root bypass patcher

## Analysis & Security
- [apktool](https://apktool.org) — Decompile/recompile APKs
- [android-forensics-guide](https://github.com/OutrageousStorm/android-forensics-guide) — Frida + static analysis
- [android-wakelock-analyzer](https://github.com/OutrageousStorm/android-wakelock-analyzer) — Battery drain analysis
- [MobSF](https://github.com/MobSF/Mobile-Security-Framework-MobSF) — Automated security testing

## System Monitoring
- [PCAPdroid](https://github.com/emanuele-f/PCAPdroid) — Network traffic capture (no root)

## Package Management
- [F-Droid](https://f-droid.org) — Open-source app installer
- [Obtainium](https://github.com/ImranR98/Obtainium) — Install from GitHub releases directly
