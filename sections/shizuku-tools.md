# Shizuku Tools

Apps that use Shizuku for elevated ADB access without root.

## System Control
- [App Manager](https://github.com/MuntashirAkon/AppManager) — full app control, uninstall, disable, component control, permissions
- [Hail](https://github.com/aistra0528/Hail) — freeze/suspend/uninstall without root
- [Shizuku Manager](https://github.com/RikkaApps/Shizuku) — the framework itself, with elegant UI

## Privacy & Monitoring
- [TrafficStats](https://github.com/eycorsican/Mio) — real-time network usage per app
- [NetGuard Shizuku](https://netguard.me/) — firewall with elevated logging via Shizuku

## System Tweaks
- [Settings Storage](https://github.com/RikkaApps/SettingsStorage) — modify system settings normally hidden
- [LSPosed](https://github.com/LSPosed/LSPosed) — Xposed framework via Shizuku/KernelSU

## Battery & Performance
- [AccA / ACC](https://github.com/MatteCarra/AccA) — charging control via Shizuku
- [Thermal Throttling Manager](https://github.com/erfanoabdi/thermal-throttle) — control thermal limits

---

> **Note:** Shizuku requires initial setup via USB debugging once, then persists systemwide. No root needed.
