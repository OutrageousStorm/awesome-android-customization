# SystemUI & Status Bar Mods

Customize the Android system UI — status bar, quick settings, lockscreen, notification shade.

## Status Bar

| Tool/Mod | Type | What it does |
|----------|------|-------------|
| **Rboard Status Bar** | Magisk module | Pixel-style status bar on any Android |
| **Iconify** | LSPosed module | Full SystemUI theming — icons, colors, animations |
| **SystemUI Tuner** | Play Store app | Unlock hidden status bar tweaks (battery %, headers only) |
| **Custom Status Bar Clock** | Xposed module | Change clock format, hide date, add emoji |
| **Navbar Customizer** | Magisk module | Hide/customize navigation buttons |

## Quick Settings Panel

| Tool | What it does |
|------|-------------|
| **QS Customizer** | Magisk + LSPosed | Reorder QS tiles, add custom tiles, change layout |
| **StatusBar Shortcuts** | Xposed module | Add direct app shortcuts to status bar |

## Lockscreen

| Tool | Details |
|------|---------|
| **Always On Display customization** | AOD-Suite (Shizuku) or via KernelSU |
| **Custom Clock Face** | LSPosed hook system clock widget |
| **Lockscreen Widgets** | Magisk DWL module (on some devices) |

## Notification Shade

| Mod | Feature |
|-----|---------|
| **Custom notification backgrounds** | Via app or LSPosed hook |
| **Notification history** | Android 13+ native, backported via Xposed |
| **Do Not Disturb icon** | Toggle in QS + Xposed theming |
