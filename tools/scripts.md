# Automation Scripts

Ready-to-run scripts that enhance Android customization without root (or minimal root).

## ADB Scripting

| Tool | Language | What it does |
|------|----------|-------------|
| [android-toolkit-scripts](https://github.com/OutrageousStorm/android-toolkit-scripts) | Python | Device info, permission audit, app extractor, debloater |
| [android-tweaks-toolkit](https://github.com/OutrageousStorm/android-tweaks-toolkit) | Python/Bash | Screen monitor, brightness scheduler, app launcher |
| [device-fingerprint-spoofer](https://github.com/OutrageousStorm/device-fingerprint-spoofer) | Python | Spoof ro.build.fingerprint, ro.product.model, ro.build.tags |
| [apk-patcher](https://github.com/OutrageousStorm/apk-patcher) | Python | Patch APKs: SSL bypass, root detection bypass |

## Dynamic Analysis

| Tool | Language | What it does |
|------|----------|-------------|
| [frida-scripts-android](https://github.com/OutrageousStorm/frida-scripts-android) | JavaScript | Frida hooks: SSL, root, crypto, HTTP, intents, SharedPrefs |
| [android-forensics-guide](https://github.com/OutrageousStorm/android-forensics-guide) | Guides + JS | APK analysis, Frida, network interception, MobSF |

## System Monitoring

| Tool | Language | What it does |
|------|----------|-------------|
| [android-wakelock-analyzer](https://github.com/OutrageousStorm/android-wakelock-analyzer) | Python | Parse bugreports, find battery drain culprits |
| [android-notification-listener](https://github.com/OutrageousStorm/android-notification-listener) | Python | Capture, dump, auto-reply to notifications |
| [android-scrcpy-tools](https://github.com/OutrageousStorm/android-scrcpy-tools) | Python | scrcpy profiles, automation scripts, multi-device |

## Privacy Hardening

| Tool | Language | What it does |
|------|----------|-------------|
| [android-privacy-hardener](https://github.com/OutrageousStorm/android-privacy-hardener) | Python/Bash | 3-level privacy automation, checker, permission revoker |
| [android-debloat-lists](https://github.com/OutrageousStorm/android-debloat-lists) | Lists + Bash | Safe debloat lists per OEM, ready-to-use shell script |

---

All tools work with **no root** (ADB only) unless otherwise noted.
