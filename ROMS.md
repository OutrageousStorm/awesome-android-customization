# 🖥️ Custom Android ROMs Comparison

| ROM | Type | Bloat | Privacy | Updates | Best For |
|-----|------|-------|---------|---------|----------|
| **GrapheneOS** | Hardened AOSP | Minimal | Excellent | Frequent | Security first |
| **LineageOS** | AOSP fork | Minimal | Good | Community | Dev devices |
| **CalyxOS** | Privacy AOSP | Minimal | Excellent | Regular | Privacy enthusiasts |
| **crDroid** | Feature AOSP | Light | Good | Frequent | Customization |
| **Evolution X** | Gaming-focused | Light | Good | Regular | Gamers |
| **PixelOS** | Pixel experience | Minimal | Good | Regular | Stock feel + features |
| **DerpFest** | Custom AOSP | Light | Good | Sporadic | Customizers |
| **Stock OEM ROM** | Official | Heavy | Poor | Device-dep | Device stability |

## Installation paths by ROM

```bash
# LineageOS (most devices)
1. Unlock bootloader
2. Flash TWRP recovery
3. Flash LineageOS zip
4. Flash GApps (optional)

# GrapheneOS (Pixel only)
1. Run web installer: https://grapheneos.org/install/web
2. Auto-patches & flashes everything

# CalyxOS (Pixel + selected others)
1. Use Calyxos installer
2. Includes microG by default
```
