# New Awesome Entries (Q2 2026)

Additions to the awesome-android-customization list:

## New Tools

- **[PCAPdroid](https://github.com/emanuele-f/PCAPdroid)** — Network traffic capture on Android without root. VPN-based packet sniffer, full traffic analysis. ⭐ 1.2K
- **[Obtainium](https://github.com/ImranR98/Obtainium)** — Install and auto-update apps directly from GitHub, GitLab, SourceForge, and more. ⭐ 3.5K
- **[Aegis Authenticator](https://github.com/beemdevelopment/Aegis)** — Open-source 2FA/TOTP manager with encrypted backups. ⭐ 4.2K
- **[OpenBoard](https://github.com/openboard-team/openboard)** — Privacy-focused keyboard — open source, no tracking. ⭐ 2.1K
- **[Organic Maps](https://github.com/organicmaps/organicmaps)** — Offline maps, fork of MAPS.ME. ⭐ 5.8K
- **[Nextcloud](https://nextcloud.com)** — Self-hosted file sync, backup, calendar, contacts. ⭐ 25K
