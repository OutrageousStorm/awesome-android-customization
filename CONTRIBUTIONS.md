# Contributing to Awesome Android Customization

Want to add your favorite app or tool? Great! Here's how.

## Adding an entry

1. Fork this repo
2. Edit README.md
3. Add your entry in the appropriate section:
   ```markdown
   - **App Name** — Description (best for X)
   ```
4. Include a link if it's a GitHub repo or Play Store listing
5. Submit a pull request

## Guidelines

- ✅ Must be actually useful (tested by you)
- ✅ Open source preferred, but paid apps okay if genuinely good
- ✅ Include a brief note about what makes it special
- ✅ Avoid duplicates — check existing entries first
- ❌ No spam or affiliate links
- ❌ No apps that require root unless specified

## Sections

- **Customization & Theming** — launchers, icon packs, themes
- **Productivity** — task managers, note-taking, etc.
- **Privacy & Security** — VPNs, firewalls, permission managers
- **Development Tools** — ADB, terminal emulators, debugging
- **Performance Tweaks** — ROM tools, kernel tweakers

## Review process

PRs are reviewed within a week. We'll ask for changes if:
- Description is unclear
- App doesn't work as described
- Duplicate exists already
- Links are broken

Thanks for contributing! 🙏
