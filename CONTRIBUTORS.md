# Contributors

This awesome list is maintained and curated by the Android community.

## How to contribute
1. Fork the repo
2. Add links with a brief description
3. Submit a PR with why this tool is awesome

## Featured contributors
- [OutrageousStorm](https://github.com/OutrageousStorm) — curator

## Community submissions
Community members have contributed:
- Device tree guides
- ADB automation scripts
- Frida exploitation frameworks
- Custom ROM comparisons
- Privacy hardening checklists

All contributions are welcome. Please ensure:
- Link is working and active
- Description is clear and honest
- Tool is genuinely useful (not spam/malware)
- GitHub projects have active maintenance or clear "archived" status
