## Customization Tools

### ADB Tools (No Root)
- **android-toolkit-scripts** — device info, permission audit, debloat, network monitor
- **android-adb-cheatsheet** — 150+ ADB commands reference
- **apk-patcher** — patch APKs for SSL pinning, root detection bypass

### Root Tools (Magisk/KernelSU)
- **android-privacy-hardener** — bulk permission revoking, hardening automation
- **android-permission-auditor** — scan all apps for dangerous permissions
- **Hail** — freeze/disable apps without uninstalling

### Dynamic Analysis
- **frida-scripts-android** — drop-in Frida scripts (SSL bypass, root bypass, crypto hooks, HTTP logging)
- **android-forensics-guide** — APK analysis, Frida, network interception guide
