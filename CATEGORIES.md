# Android Customization Categories

## ROMs
- LineageOS — stock-like AOSP, regular updates, 400+ devices
- GrapheneOS — privacy-hardened Pixel-only
- CalyxOS — AOSP + microG, privacy focus
- Evolution X — feature-rich customization
- crDroid — gaming + performance optimized
- /e/OS — de-Google movement fork

## Launchers
- Nova Launcher — most feature-rich
- Lawnchair — open source AOSP-based
- Niagara — minimal, gesture-based
- Poco Launcher — clean lightweight
- Microsoft Launcher — Office integration

## Customization (no root)
- Substratum — theme engine (deprecated but still works)
- One Hand Operation+ — edge gestures
- Macrodroid — automation without root
- Tasker — advanced automation
- KWGT — custom widgets

## Customization (with root/Magisk)
- Magisk modules (LSPosed, Shamiko, etc.)
- KernelSU modules
- Xposed modules (legacy)
- Substratum with Andromeda

## Performance & Battery
- Greenify — app hibernation
- AccA — charge controller
- Universal GMS Doze — aggressive GMS dozing
- ForceDoze — aggressive background restriction

## Privacy & Security
- NetGuard — no-root firewall
- TrackerControl — block trackers
- Orbot — Tor proxy
- Molly — hardened Signal
- MicroPinDroid — DNS monitor

See full list: https://github.com/OutrageousStorm/awesome-android-customization
